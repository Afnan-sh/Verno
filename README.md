# Verno - Voice Agentic Coding Agent

Verno is a VS Code extension that brings voice-controlled AI-powered coding agents to your editor. It enables you to use voice commands to interact with a suite of specialized coding agents that handle various development tasks.

## Features

- **Voice Recording**: Start and stop voice recordings directly from your editor
- **AI-Powered Agents**: Leverage specialized agents for code generation, debugging, refactoring, and more
- **Code Generation**: Generate code snippets and complete functions using natural language
- **Code Review**: Get AI-powered code reviews and suggestions
- **Debugging**: Troubleshoot issues with AI assistance
- **Documentation**: Auto-generate documentation for your code
- **Test Generation**: Create unit tests automatically
- **Refactoring**: Suggest and apply code refactoring improvements

## Commands

- `Verno: Process User Input` - Process voice input or text commands
- `Verno: Start Recording` - Begin voice recording
- `Verno: Stop Recording` - End voice recording
- `Verno: Manage Agents` - Configure and manage AI agents

## Requirements

- VS Code 1.80.0 or higher
- Node.js 18 or higher

## Getting Started

1. Install the extension
2. Open it in the side bar
3. Add your api key
4. Speak or type your command (e.g., "Generate a function to sort an array")
5. The extension will process your request and display results

## Extension Settings

Configure Verno through VS Code settings to customize:
- LLM provider selection
- Voice recording preferences
- Agent behavior and capabilities

## Known Issues

- Voice recording requires appropriate permissions on your system
- Some LLM providers require API keys to be configured
